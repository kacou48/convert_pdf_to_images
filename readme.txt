PYTHON 3.9

This little script allows you to convert  pdf to multiple images

Open your command prompt, activate a virtual environment, and install the independences:
pip freeze > requirements.txt

OR Install pdf2image and pillow: pip install pdf2image, pillow

2) download poppler (in my case it is a version of window) then you extract the file
download link: https://blog.alivate.com.au/poppler-windows/
